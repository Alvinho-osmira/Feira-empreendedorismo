Como instalar uma distribuição Linux

Confira o passo a passo em vídeo (em português):


⚠️ Recomendação: Use um pendrive com pelo menos 8 GB de espaço (preferencialmente 16 GB ou mais) para criar o instalador do Linux

✅ 1. Baixe a imagem ISO: Acesse o site oficial da distribuição (como Ubuntu, Linux Mint, Fedora).

✅ 2. Crie um pendrive bootável: Use programas como Rufus (Windows) ou balenaEtcher (Linux) para gravar a ISO.

✅ 3. Inicie pelo pendrive: Reinicie o computador, entre na BIOS/UEFI e altere a ordem de boot.

✅ 4. Siga o assistente de instalação: Escolha idioma, fuso horário, tipo de instalação (usar disco inteiro ou particionar manualmente).

✅ 5. Finalize: Aguarde a instalação e reinicie. Remova o pendrive quando solicitado.

Comandos básicos do terminal Linux

Aprenda os comandos essenciais para iniciantes:


✅ 1. Navegação: cd (entrar em diretório), ls (listar arquivos), pwd (caminho atual).

✅ 2. Manipulação de arquivos: cp (copiar), mv (mover/renomear), rm (remover).

✅ 3. Permissões: chmod (alterar permissões), chown (alterar dono).

✅ 4. Pacotes: sudo apt update, sudo apt install nome-do-pacote (para Debian/Ubuntu).

✅ 5. Ajuda: man comando ou comando --help.

Como instalar e remover programas no Linux

Confira o passo a passo em vídeo:


Linux (Todas as distros) ⤵︎


✅ APT (Debian/Ubuntu): sudo apt install nome para instalar, sudo apt remove nome para remover.

✅ DNF/YUM (Fedora/Red Hat): sudo dnf install nome ou sudo yum install nome.

✅ Snap: sudo snap install nome.

✅ Flatpak: flatpak install flathub org.app.Nome.

✅ Dica: Sempre atualize com sudo apt update && sudo apt upgrade (ou equivalente).
